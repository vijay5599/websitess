## 🚀 How to Run Each Website

### For any of the 3 websites:

```bash

cd /Users/vijay5599/Developer/Scrape/websitess/{website_name}

# 2. Install dependencies
npm install

# 3. Create .env.local file with your OpenAI API key:

# 4. Run development server

# 5. Open http://localhost:3000 in your browser
```

## 💾 Data Storage

All booking/enrollment/service data is stored in browser localStorage:
- Real Estate: `propertyVisits`
- Coaching Center: `enrollments`
- Car Service: `serviceBookings`

To view stored data in browser console:
```javascript
// Real Estate
console.log(JSON.parse(localStorage.getItem('propertyVisits')))

// Coaching
console.log(JSON.parse(localStorage.getItem('enrollments')))

// Car Service
console.log(JSON.parse(localStorage.getItem('serviceBookings')))
```
